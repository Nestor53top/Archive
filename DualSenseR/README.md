# DualSenseR

Fork of [DualSenseX](https://github.com/Paliverse/DualSenseX) with renamed branding.

## About

Bring your DualSense Controller to life on PC with support for Adaptive Triggers!

Emulate with the following while also getting the functionality of a DualSense Controller:
- Xbox360
- DualShock 4
- OFF (for no Emulation, reading only inside app)

## Features

- Adaptive Triggers with 19+ modes
- Haptic Feedback / Audio Haptics
- Controller LED customization (Static, Rainbow)
- Touchpad to Mouse mapping
- Sense2Key (Controller to Keyboard mapping)
- UDP Server for game mods
- Text File reader for game mods
- Auto-connect controller
- Multi-language support (30+ languages)

## Requirements

- Windows 10/11 (64-bit)
- [ViGEmBus Driver](https://github.com/ViGEm/ViGEmBus/releases/latest)
- [HidHide Driver](https://github.com/ViGEm/HidHide/releases/latest)
